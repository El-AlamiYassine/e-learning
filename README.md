📚 E-Learning Web App

Une application web moderne dédiée à l’apprentissage en ligne, permettant aux utilisateurs d’accéder à des cours interactifs, suivre leur progression et améliorer leurs compétences à leur propre rythme.

🚀 Fonctionnalités principales :

Authentification sécurisée (inscription / connexion)
Catalogue de cours avec catégories
Lecture de contenu (vidéos, textes, quiz)
Suivi de progression en temps réel
Système d’évaluation et de quiz
Tableau de bord utilisateur personnalisé

🛠️ Technologies utilisées :

Frontend : React JS
Backend : Spring boot
Base de données : MySQ
API REST

🎯 Objectif :
Offrir une plateforme intuitive et accessible pour faciliter l’apprentissage en ligne, que ce soit pour les étudiants, les formateurs ou les autodidactes.
